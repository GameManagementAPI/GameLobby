# GameLobby
GameLobby is an extension plugin for [GameManager](https://github.com/GameManagementAPI/GameManager)
that provides core lobby functionality for games built on the GameManager ecosystem. It adds player-friendly lobby mechanics like queues, map voting, and join signs

### Features:
- Lobby spawn points 
- Join signs for games 
- Map voting system 
- Custom queue system 
- Custom spectator handling 
- Fully customizable messages and sign layouts

### Requirements
GameLobby depends on GameManager and will not function without it.

➡️ Install the latest release of GameManager before starting the server:
https://github.com/GameManagementAPI/GameManager/releases/latest

### Installation

1. Download the [latest GameLobby release](https://github.com/GameManagementAPI/GameLobby/releases/latest).
2. Place the jar file into your server plugins folder.
3. Ensure GameManager is installed and configured.
4. Restart your server.
5. Edit `config.yml` to customize settings if needed.

### Configuration
The plugin generates a `config.yml` file where you can customize lobby behavior.

### Formatting
This plugin supports MiniMessage formatting for text components, allowing full color and styling customization.
